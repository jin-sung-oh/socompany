# 2026-03-08 작업 기록
## 완료된 작업
- Animal Office를 `48px 타일`, `grid index`, `collision matrix`, `Y-sorting` 기준의 Gather형 spatial office 구조로 재설계
- 오피스 데이터를 퍼센트 좌표 레이아웃에서 `floor map / object list / seat map / collision map` 기반 타일맵 데이터로 교체
- `OfficeDashboard2D`를 입력/카메라/대화 오케스트레이션 역할로 줄이고, `OfficeMap`, `OfficeTile`, `OfficeCharacter`, `OfficeHUD`, `OfficeConversationPanel`, `spatialPrompt`로 atomic 분리
- 플레이어를 `WASD/방향키` 4방향 타일 이동 + `0.2s` 선형 이동감 기준으로 재구성하고, 카메라는 플레이어 중심 follow 방식으로 정리
- NPC 이동도 grid waypoint + BFS 기반 한 칸 이동으로 바꿔 플레이어와 같은 월드 좌표계를 공유하도록 보정
- proximity 반경을 `2타일`로 맞추고, 근처 NPC가 있을 때 플레이어 위에 `Press [Space] to talk` 툴팁과 하단 대화창 슬라이드업 흐름 추가
- 대화 시 `Species & Role`, `Personality`, `Dialogue Style`, `Current_Location`, `WorldState`, `User_History`를 넣고 `{"message","action"}` JSON을 요구하는 spatial prompt 경로 추가
- 좌측 Active Animals, 우측 Mini-map, 하단 Conversation 패널을 갖는 Gather형 HUD 오버레이 1차 구현
- `Agent` 공용 타입과 기본 에이전트 데이터에 `personality`, `dialogueStyle` 필드 추가
- Tailwind CSS 도입을 시도했지만 `registry.npmjs.org` DNS 실패와 pnpm store 불일치로 설치가 막혀, 이번 턴은 기존 CSS 스택으로 동일 구조를 구현
- 플레이어 이동을 자유 이동에서 Gather식 상하좌우 4방향 그리드 스냅 이동으로 전환하고, 가장 최근 방향키 기준으로 타일 단위 이동하도록 보정
- 플레이어가 바라보는 앞 타일을 기준으로 문/의자/NPC 상호작용을 판정하고, 발치 링 + 앞 타일 하이라이트 레이어로 상호작용 가능 영역을 시각화
- 근접한 NPC와의 거리를 `opacity/volume` 강도로 환산하는 spatial chat foundation을 추가하고, 말풍선과 HUD에 대화 강도 수치를 연결
- 오피스 카메라를 수동 팬에서 Gather식 player-centered camera follow로 되돌리고, stage transform을 플레이어 중심 추적 기준으로 정리
- 오피스 이동 규칙을 공용 `officeNavigation` 레이어로 분리해 플레이어와 NPC가 같은 벽/문/충돌 규칙을 공유하도록 정리
- 문 데이터에 room zone, hall/room anchor를 추가해 단순 장식이 아니라 실제 통과 포인트로 동작하도록 확장
- 플레이어 이동에 NPC 충돌 반경을 적용하고, 문 근처에서 `E/Enter`로 방 안팎을 오갈 수 있는 doorway pass-through 상호작용 추가
- HUD 힌트를 `Meeting Room 들어가기/나가기`처럼 실제 room transition 문구로 갱신하고, 문 통과 시 로그가 남도록 보강
- 에이전트 이동 로직을 벽 회피 + room waypoint 기반으로 변경해 focus booth, meeting room 같은 구획 공간도 문을 따라 드나들 수 있게 보강
- 에이전트 wandering target 생성 시 벽 내부나 다른 캐릭터와 겹치는 좌표를 피하도록 이동 후보 생성 로직 개선
- focus booth 문 구간에서 corridor와 room zone이 겹치며 왕복하던 문제를 room 우선 zone 판정으로 수정
- 오피스 맵을 viewport/stage 구조로 바꾸고 마우스 드래그 기반 수동 카메라 팬 및 시점 초기화 버튼 추가
- 책상과 의자를 모니터, 키보드, 머그컵, 회전 의자 형태로 다시 그려 오피스 가구 느낌을 더 사실적으로 보강
- 플레이어가 에이전트 가까이 가면 자동으로 반응하는 proximity 링/말풍선/HUD 피드백과 근접 로그를 1차 구현
- 기본 카메라 줌을 제거해 화면이 스스로 내려가거나 밀리는 느낌을 줄이고, 수동 팬만 남도록 보정
- door lane 내부에서는 room/hall 앵커 중 한쪽으로 끝까지 통과하도록 고정하고, 문 근처 충돌 반경을 완화해 문 앞 왕복 현상 추가 보정
- 벽, 유리문, 라운지/회의실 톤을 더 실제 오피스 재질에 가깝게 다시 스타일링
- `DashboardView` 대화 패널의 `scrollIntoView`를 내부 `scrollTop` 제어로 바꿔 페이지 전체가 아래로 끌려가던 자동 스크롤 문제 수정
- `pnpm build` 성공 확인
