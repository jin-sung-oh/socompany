# Ollama 설치 및 설정 가이드

KAFI는 로컬 LLM을 활용하여 보안과 성능을 모두 잡은 AI Agent 서비스를 지향합니다. 현재 KAFI의 기본 모델은 **Ollama**를 통해 실행되는 **llama2** 모델입니다.

## 1. Ollama 설치하기

Ollama는 로컬에서 다양한 대규모 언어 모델(LLM)을 간편하게 실행할 수 있는 오픈소스 도구입니다.

1.  [Ollama 공식 웹사이트(ollama.com)](https://ollama.com/download)에 접속합니다.
2.  사용 중인 운영체제(macOS, Windows, Linux)에 맞는 설치 파일을 다운로드합니다.
3.  다운로드된 파일을 실행하여 설치를 완료합니다.

## 2. 기본 모델 다운로드 (llama2)

KAFI를 실행하기 전, 기본적으로 사용하는 `llama2` 모델을 다운로드해야 합니다.

1.  터미널(Terminal) 또는 명령 프롬프트(CMD)를 엽니다.
2.  아래 명령어를 입력하여 `llama2` 모델을 다운로드하고 실행합니다.
    ```bash
    ollama run llama2
    ```
3.  다운로드가 완료되면 터미널에서 카피바라와 대화하듯 테스트해 볼 수 있습니다.
4.  테스트를 마치려면 `/bye`를 입력하거나 `Ctrl + D`를 눌러 종료합니다.

## 3. KAFI와 연결 확인

1.  Ollama 앱이 백그라운드에서 실행 중인지 확인합니다 (메뉴 바/트레이 아이콘).
2.  KAFI 앱을 실행합니다 (`pnpm dev`).
3.  위젯 또는 대시보드에서 채팅을 시도했을 때 응답이 오면 정상적으로 연결된 것입니다.

## 💡 팁
- 다른 모델(예: `mistral`, `gemma`, `phi`)을 사용하고 싶다면 `ollama pull [모델명]` 명령어로 미리 다운로드해 둘 수 있습니다.
- 현재 KAFI는 기본적으로 `llama2`를 호출하도록 설정되어 있으나, 추후 설정 화면에서 변경 기능을 제공할 예정입니다.
